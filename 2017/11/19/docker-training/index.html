    docker demo仓库地址

    git clone https://gitee.com/dockerf/docker-training.git

    构建第一个docker镜像：

    cd /docker-training/centos7/
    docker build -t csphere/centos:7.1 .
    //build:意思是通过dockerfile来生成一个docker镜像，t：镜像文件名。

    查看生成镜像

    docker images
    cd

    ###通过docker镜像生成容器

    docker run -d -p 2222:22 –name csphere/centos:7.1

    ###查看所有的container

    docker ps -a
