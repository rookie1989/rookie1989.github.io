    创建bash_aliases文件

    cd
    vim .bash_aliases
    alias gits=’git status’

    通过source命令重新执行命令生效

    source .bash_profile
    source .bash_aliases

    现在你就可以在命令行窗口直接输入gits代替git status命令了
